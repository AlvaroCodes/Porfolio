### Tecnologías 
AstroJS · HTML · CSS · JavaScript · TypeScript

Algunos detalles técnicos destacados:

**TypeScript:**  

Utilizo TypeScript para contar con tipado estático, lo que mejora la detección temprana de errores y facilita el mantenimiento del código, haciendo el desarrollo más robusto y escalable.

**AstroJS:**  

La base de este proyecto, que me permitió desarrollar una web rápida y moderna.

- Uso de content: La carpeta content en Astro organiza y gestiona fácilmente todo el contenido, integrando archivos Markdown y MDX de forma ágil.

- Enrutamiento Dinámico: Implementación de rutas dinámicas para generar páginas basadas en colecciones de artículos, facilitando la creación de contenido dinámico.

- Transiciones de Cliente (View Transitions): Uso de transiciones que mejoran la experiencia de navegación.

**Despliegue con Cloudflare:**  

Gracias a su CDN global, Cloudflare reduce la latencia, mejora los tiempos de carga y ofrece protección avanzada, garantizando alta disponibilidad y seguridad.
