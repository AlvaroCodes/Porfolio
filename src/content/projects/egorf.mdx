Desarrollo y mantenimiento del sitio web oficial del European Golden Oldies Rugby Festival (EGORF) 2024, un evento internacional que reunió a más de 2.500 jugadores de rugby veteranos en Sevilla.

🔗 [Web - EGORF 2024](https://egorfsevilla2024.es/)

📅 Fecha: Mayo 2023 - Junio 2024

### Tecnologías 
React · HTML · CSS · JavaScript · NodeJS · MySQL

## Responsabilidades

**Desarrollo de funcionalidades**  
- Desarrollé la aplicación web multilingüe en cuatro idiomas utilizando React.
- Creé una aplicación para generar el contenido de manera dinámica con NodeJS.
- Gestioné la información y la base de datos con MySQL, asegurando integridad y eficiencia en el almacenamiento.
- Mantuve trato directo con clientes como Veteranos de Rugby Sevilla y El Corte Inglés, proporcionando soluciones técnicas adaptadas a sus necesidades y garantizando una comunicación efectiva.
